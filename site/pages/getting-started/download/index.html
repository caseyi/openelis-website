Download & install OpenELIS Global

OpenELIS is free and open-source under the Mozilla Public License. The recommended way to run it is with Docker — up and running out of the box.

Docker setup guide

Two ways to install

Docker — recommended

The maintained, out-of-the-box deployment. Install Docker and Docker Compose, then bring up a full OpenELIS stack (app, database, FHIR layer) with a single command. This is the path we test and update with every release.

Docker setup guide →

Offline installer

For sites without reliable internet, download the offline installer from the release assets and follow the offline setup guide. Note: the offline installer is updated less frequently than the Docker images.
