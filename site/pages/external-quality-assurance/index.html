Why it matters

External quality assessment is a cornerstone of laboratory accreditation — but running a proficiency-testing program normally means dedicated PT software, and participating means yet another portal. For a national reference laboratory or an EQA provider, being able to create and distribute panels from the same LIMS the network already uses is a genuine differentiator: one system, one dataset, one accreditation story.

Both sides of the EQA workflow

EQA provider mode

For reference labs and PT coordinators: create rounds and panels with expected values and acceptable ranges, enroll and manage participating labs, track sample distribution with chain of custody, analyze network performance, and issue certificates and reports.

Participating lab mode

For clinical and district labs: receive and log PT samples, submit results before the deadline through the familiar OpenELIS interface, compare against expected values and peers after cutoff, track historical performance, and document corrective actions.

Everything an EQA program needs

Flexible panels

Any test type — chemistry, hematology, microbiology, serology, molecular — with expected values, ranges, and scoring criteria.

Distribution & logistics

Track shipments from dispatch to receipt, confirm sample integrity, and manage courier information.

Statistical scoring

Automatic z-scores, bias, and performance metrics with robust statistics that handle outliers.

Performance trending

Track labs across rounds, flag declining performance before it becomes critical.

Corrective actions

Document root-cause analysis and corrective actions linked to specific events.

Accreditation-ready

Participation records, trending, and documented actions in the formats auditors expect for ISO 15189, SLIPTA, and ISO 17043.
